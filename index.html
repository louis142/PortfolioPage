My Projects

True Picture

Console app for students and tutors to bridge the gap on mental health. The project had a particular focus on how the app was being developed reinforcing techniques like identifying personas and user stories to build a portfolio of who would be using the app and their requirements.

Web Server

Designing a web client from the ground up, starting with the SQL database and moving on to a program that handles http network requests to that table. This project involved a lot of moving parts from understanding how to normalise my database to coding each of the catch cases for a bad request.

Neural Network vs Regression Model

Weighing the performance and accuracy of several AI models. Considering factors like over-fitting and under-fitting, allowing me to understand the mechanisms and limits of these models to select the best for the job.

Cinema ePOS

Console App for a cinema that handles transactions and movie screenings. The process of building the project instilled many object oriented principles allowing me to experience the necessity of them, such as inheritance, through problem solving.

Network Management On The Cloud

Building a virtual network with accompanying virtual machines on Azure to understand the significance of network security systems. Deciding the security levels and using network utility tools to determine if each machine is secure enough taught me to consider scenarios which would leave the whole network vulnerable.

E-Speed: Team Design Project

Developing a mock prototype for an electric bike company with an accompanying app. Strengthening team skills and encouraging open dialogue to develop a team pitch we are all proud of.
